# Sistema de Login

Projeto simples de autenticação com cadastro e login de usuários, utilizando LocalStorage para simular um banco de dados no navegador.

## Funcionalidades

- Cadastro de novos usuários
- Login com validação de credenciais
- Persistência de dados com LocalStorage
- Redirecionamento entre páginas
- Estrutura organizada por páginas e scripts

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)

## Estrutura do projeto

css/
js/
pages/
├── auth/
└── home/


## Objetivo

Projeto desenvolvido para praticar conceitos de front-end e lógica de autenticação sem uso de backend.

## Observação

Este projeto não utiliza criptografia ou segurança real, sendo apenas para fins educacionais.
